# Nestjs Oidc-protect

A NestJS OpenID Connect library that can validate a token and protect routes with Auth and Role Guards

## Installation

install with npm:

```bash
npm install @flowcore/nestjs-oidc-protect
```

or yarn:

```bash
yarn add @flowcore/nestjs-oidc-protect
```
